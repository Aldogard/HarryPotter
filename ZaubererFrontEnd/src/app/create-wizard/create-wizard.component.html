Rules for the Creation of a wizard:

  1. There can be only one Voldemort
  2. There can be only one Dumbledore
  3. Every name can be used only once
  4. A wizard type must be chosen
  5. The name has to be at least 3 characters long
  6. Healthpoints cannot be smaller than 1
  7. Healthpoints cannot be larger than 100
  8. A description must be written, but it cannot extend 100 characters
